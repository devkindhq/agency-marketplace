# Security Policy

## Responsible Use

The tools in this repository (WhatWeb, theHarvester) perform network reconnaissance. You must only scan or harvest information from domains and hosts that you own or have explicit written permission to test. Unauthorised scanning may violate computer fraud laws in your jurisdiction.

## Reporting a Vulnerability

If you discover a security vulnerability in this repository, please email ali@devkind.com.au with:
- A description of the vulnerability
- Steps to reproduce
- Potential impact

Please do not open a public GitHub issue for security vulnerabilities. We will respond within 5 business days and coordinate a fix before any public disclosure.
